# TRA_Codeline

A comprehensive learning project covering network programming, system administration, and infrastructure automation in Python.

## Project Structure

### Daily Lessons (Day 1 - Day 7)

- **Day 1**: IP Validation and Basic Testing
  - `ip_validator.py` - IP address validation utilities
  - `test.py` - Unit tests for Day 1

- **Day 2**: Network Operations
  - `day2.py` - Day 2 exercises
  - `test2.py` - Unit tests for Day 2

- **Day 3**: Advanced Testing
  - `t3.py` - Day 3 exercises

- **Day 4**: (In development)

- **Day 5**: (In development)

- **Day 6**: System Operations
  - `Day6.py` - Day 6 exercises

- **Day 7**: Network Scanning
  - `network_ping_scan.py` - Network ping and scanning utilities

### SSH & Authentication

- **SSH/** - SSH key authentication and management
  - `id_rsa_paramiko` - Private SSH key
  - `id_rsa_paramiko.pub` - Public SSH key
  - `task5_ssh_key_auth.py` - SSH key authentication examples

### Final Project

- **Final Project/** - Capstone project
  - `final_Project.py` - Main project file

### Utilities

- `client.py` - Client utility for various operations
- `t3.py` - Additional test/utility script

## Getting Started

### Prerequisites

- Python 3.x
- Required packages (if any) can be installed via pip

### Running the Scripts

```bash
# Run specific day exercises
python Day\ 1/ip_validator.py

# Run unit tests
python Day\ 1/test.py

# Run SSH authentication examples
python SSH/task5_ssh_key_auth.py

# Run network scanning
python Day\ 7/network_ping_scan.py
```
